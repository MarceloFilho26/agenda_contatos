# Agenda de contatos

Progama em java que consiste na criação de uma agenda para contatos.

Funcionalidades:
* Adicionar contato;
* Exibir os contatos;
* Buscar os contatos;
* Remover contato;

PS: Esse programa ainda passará por atualizações, portanto é normal que encontrem erros.
